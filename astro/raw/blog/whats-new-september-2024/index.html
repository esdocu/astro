$100,000 Astro Ecosystem Fund Fall Update

Congratulations to the latest recipients of Astro’s ecosystem grants! Check out these projects that have made a huge impact:

  • 🖌 @hippotastic ($10,000) for work on the Expressive Code advanced syntax highlighter that powers many Astro websites, and is built-in to Starlight by default!

Tips & Tools

The latest community-built utilities and integrations to help you build with Astro.

  • Full dev UI - The Astro UI library to build content sites with prebuilt components and blocks.
  • Bag of Tricks: <ElementCrossing /> - For smoother and smarter cross-document view transitions.
  • Astro dts builder - For generating TypeScript declaration files for Astro Integrations.
  • astro-minify-html - Minifies Astro’s generated HTML, including inline scripts & styles.
  • zastro-service-worker - An Astro integration that adds service worker functionality to your Astro project, powered by Workbox.
  • astro-turnstile - An Astro integration that allows you to add a turnstile to your Astro site.
  • GitHub repo loader - A content collections loader for GitHub repos.
  • Bag of Tricks: <TurnSignal /> - Create distinct animations for forward and backward navigation.
  • Astro Test Utils - Tooling for testing your own Astro Integrations and libraries.
  • Cut-Short - A simple tool for ending requests early with a custom response.
