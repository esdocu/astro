Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. 🛠

  • react-router-astro - A starter template for creating a server-rendered SPA using React Router and Astro.
  • astro-rerun-island - An integration that reruns your persistent server islands on astro:after-swap to refresh your content without showing the user unnecessary loading screens.
  • astro-meta - Easily add meta tags to your Astro app.
  • astro-path-helpers - An Astro integration that generates type-safe path helper functions for your Astro routes.
  • astro-ai-robots-text - An Astro integration to disallow AI scrapers in robots.txt.
  • s3-compatible-loader - A content loader to turn an S3-compatible object storage into a CMS that works well with Obisidian’s Remotly Save.
  • astro-gemtext - An Astro integration that lets you use .gmi (Gemini protocol document format) files as pages in your Astro site.
  • astro-itchio - An Astro integration that automates your deployment to Itch.io.
  • Astro Clerk loader - Access a selection of Clerk Backend APIs by simply providing the method name.
  • Astro Plausible loader - Provides a Plausible content loader for Astro’s content layer. You can access the Stats API v2 to view historical and real-time stats of your website.
  • 🌟 starlight-markdown-blocks - Extend Starlight’s Markdown asides syntax with custom (:::block) types.
  • 🌟 starlight-toc-overview-customizer - Enhance Starlight’s table of contents with customizable overview title
  • reveal-animation-handler - A responsive text animation system for Astro that intelligently detects and respects natural line breaks. This utility provides dynamic, viewport-aware animations with multiple animation types and customizable options while maintaining responsive design across all devices.
  • astro-appwrite-demo - A demo of using Appwrite on Astro, allowing end-users to create an account and update their username on the account page.
  • astro-d1-magiclink - Magic link auth based on Lucia-auth, Astro Actions, and Cloudflare D1.
  • @inox-tools/content-utils - Utilities for your Content Collections, now providing Git information for any file-based content loader.

… and some built with Astro:

  • WorksForMe - Enter a time and a time zone(s) and get a shareable link so everyone around the world knows exactly when that meeting is!
  • Astro UI - A Bitshares UI built using Astro, React and Electron.
