Tips & Tools

The latest community-built utilities and integrations to help you build with Astro.

  • astro-react-i18next - Integrates i18next and react-i18next seamlessly into your Astro website to provide robust i18n support for React components.
  • Astro Resend - Easily send email notifications for form submissions using Resend in an Astro project.
  • DeWP - Use your WordPress data in Astro projects
  • starlight-site-graph - A Starlight plugin adding a graph component to the site’s right-sidebar.
  • Astro NPM Loader - Load information about NPM packages into a content collection.
  • Local AstroDB Viewer - View and interact with .db files directly in Visual Studio Code.
  • Astro Mail Obfuscation - Protect email addresses and phone numbers in your Astro app’s source code from bots using XOR-based obfuscation.
  • react-alpine-store - Allows for syncing between reactive state across React and Alpine’s $store. See more information about Alpine’s $store here.
  • Expressive Code Color Chips - A plugin to add color previews to your CSS code examples
  • Astro AO3 loader - A AO3 data loader for Astro using the experimental content layer and AO3.js to load data from AO3 to your Astro website.
  • Portal Gun - Server-side Portals for adding HTML elements outside of the component that defines them.
