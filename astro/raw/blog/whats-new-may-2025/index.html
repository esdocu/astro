Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. 🛠

  • Astro Prettier Response - This package injects a middleware that runs the Prettier formatter for all output to make it unminified and pretty.
  • Astro Strapi Blocks - Astro blocks for Strapi Rich Text Block field.
  • Piny Tailwind Visual Editor - A VSCode extension that lets you visually edit Tailwind CSS right in your React and Astro files.
  • Astro Image Dithering - Dither your images before serving them to your audience for quicker image loading times combined with a lower carbon footprint.
  • Remark Auto Diff - A remark plugin providing an alternative syntax to author diff code blocks in Markdown.
  • Schema-driven Astro Starter Kit - A production-ready starter kit that brings structured data, SEO optimization, and schema-driven development to your Astro projects.
  • Astro Loader i18n - An Astro glob content loader for i18n files and folder structures.
  • WebApp Astro PWA - A ready-to-use Astro component library for adding Progressive Web App (PWA) support to your Astro projects.
  • Astro Simple Modal - A simple and highly customizable modal for Astro — minimal setup, full styling control.
  • Astro Maintenance - Easily add maintenance and coming soon pages to your Astro site.
  • Astro Iubenda - This Astro integration fetches and provides Iubenda Privacy Policy, Cookie Policy, and Terms & Conditions content for your Astro project.
  • Light-Auth - A lightweight, secure, and easy-to-implement authentication framework with an Astro Hello World example.
  • Astro Wordpress ACF Generator - A php code generator for WordPress Advanced Custom Fields (ACF).
  • Astro Daisy Theme Toggle - Component to add a dark and light theme to your AstroJS and DaisyUI project.
  • Astro OG Images - Yet another Astro integration to generate Open Graph images. More lightweight than most alternatives.
  • Astro Starter Kit: Component Package - A template for an Astro component library. Use this template for writing components to use in multiple projects or publish to NPM.
  • Flotiq Astro SDK - Dedicated Astro collections generator for Flotiq Headless CMS.
  • Astro Mailer - An Astro integration that enables you to send email notifications via any SMTP service using Nodemailer.
  • Astro Resend - An Astro integration that allows you to easily send email notifications for form submissions using Resend in an Astro project.
  • Remark Livecodes for Astro - An open-source client-side playground with support for Astro.
  • 🌟 Starlight Custom Navigation - Enhance your documentation site with custom page navigation features, including support for keyboard shortcuts.
  • 🌟 Starlight Code Block Fullscreen - Add fullscreen toggle functionality to Expressive Code blocks in your documentation.
  • Astro Adapter Nekoweb - An adapter for Astro that deploys your app automatically on Nekoweb.
  • Astro PlantUML - An Astro integration for rendering PlantUML diagrams in your markdown files.
  • Astro Tips - Customizable tip and notice blocks for Astro with 18 built-in styles and auto dark mode support.
  • Vemetric Astro SDK - Track your data via the Astro SDK for Vemetric.
  • Astroboard - Transform your Umbraco CMS with a dynamic dashboard.
  • Inox Tools Cut-short - An Astro integration that lets you stop processing a request instantly and send back a custom response, now with a cancelPrerender option to dynamically skip the generation of a specific page. Instead of rendering a 404 page, this utility ensures the page is completely omitted from your final static output.

… and some built with Astro:

  • Astro Life in Weeks - An Astro project to create your own “Your Life in Weeks”, an interactive visualization where each rectangle represents a week of your life.
  • Ubikute Relay - A new Status page for communities and organizations to selfhost, with frontend built on Astro.
  • html2astro - Drag & drop your HTML. Instantly convert to an Astro project.
  • Historical Timeline Calculator - Create and export a graphic of historical timelines.
