Latest releases

Some of the latest community-built integrations to help you build with Astro. 🛠

  • Backstro Email Compose email templates with Astro components — a direct port of react-email component to native .astro syntax.
  • Review Loop - A dev-only annotation overlay for reviewing rendered sites and feeding corrections to AI coding agents.
  • Astro Techradar - An interactive, fully static technology radar built with Astro. Track technology adoption decisions with a visual radar chart — driven entirely by Markdown files.
  • NPM Astro Render Component - Astro component renderer. Zero configuration. Produces clean HTML strings directly in Node.js without any DOM environment.
  • Lexington SEO - Render SEO tags in your page <head> using a single AstroSeo component.
  • Astro Markdown Export - An Astro integration that exports content-collection Markdown files at build time and enhances their frontmatter with optional source URLs and extra metadata for bots, crawlers, AI agents, and other tools.
  • Astro Reveal - Beautiful scroll-reveal animations for Astro with zero JavaScript in production. GPU-accelerated, performant, and accessible.
  • Neokit - An Astro component library that automatically injects Tailwind + DaisyUI into your project — no manual CSS imports needed.
  • Astro Bundle Budget - Build-time JS/CSS bundle size budgets for Astro. Inspect every asset and page payload at the end of astro build — and optionally fail CI when you exceed your thresholds.
  • LCARS Stylus - LCARS inspired design system built with Stylus.
  • Astro Icon Sprite - Build-time SVG compilation into a runtime sprite sheet.
  • Astro Stacks - A Laravel Blade-style @stack / @push / @pushOnce system for Astro SSR.
  • Astro MD Alternate - An Astro integration to serve Markdown versions of content pages for AI agents and LLMs.
  • Astro Securitytxt - An Astro integration that automatically generates an RFC 9116 compliant security.txt file during your site build.
  • Astro Class Shortener - An Astro integration to minify and mangle CSS class names across HTML, CSS, and JS files.
  • Astrobit - A thin framework for running MoonBit components as Astro islands.
  • JAAMD - Just Another Astro Markdown — remark plugins, client-side enhancements and styles as a single Astro integration.
  • Agent Markup - Build-time llms.txt, JSON-LD, Markdown mirrors, AI crawler controls, and validation.
  • CoCart Headless SDK - Build headless WooCommerce storefronts through its API instead of using PHP templates.
  • Astroscope Hyperspace - Pre-compressed in-memory static file serving for Astro SSR. Save space, serve from hyperspace.
  • GreenLight VSCode extension - A visual drag-and-drop HTML editor for Markdown, HTML, and Astro components.

CMS/Text Editors

  • Astro MD Editor - If you store content in src/content, this gives you a local editor UI focused on fast Markdown/MDX editing plus schema-driven frontmatter controls, including custom types like image/color/icon.
  • Galaxy Brain - A starter template for publishing your Obsidian vault as an interactive 3D knowledge graph website.
  • Astro Blocks - A block-first CMS. Pages, menus, settings and uploads stored in JSON, with your own Astro components as blocks.
  • Decap CMS OAuth Astro - Astro integration for Decap CMS (or any compatible CMS like Sveltia) with custom OAuth backend.

Content loaders

  • AT Astro Loader - AT Protocol-powered loaders for Astro content collections.
  • Astro Loader Sveltia CMS - A Sveltia CMS integration for Astro 6 that serves the Sveltia CMS admin UI and provides a content loader for Astro’s content collections.

Dev Toolbar apps

  • Astro Image Inspector - Astro Dev Toolbar app that lets you inspect images on hover during local development: see actual size, render size, loading mode, and common warnings (upscaled, oversized source, missing dimensions).
  • Instruckt Astro - A dev toolbar app for visual annotation feedback for AI coding agents in Astro. Click on any UI element, leave feedback with optional screenshots, and let AI agents read and fix issues.
  • Astro Toolbar UA - Astro dev toolbar app for switching user agent strings.
  • Astro Caliper - An Astro dev toolbar app designed to assist developers in converting designs to code with visual aids and inspection tools directly within the browser.
  • Astro Dev Toolbar JSONLD - An Astro Dev Toolbar app that detects, displays, and validates JSON-LD structured data on your pages during development.
  • Astro Env Inspector - A dev toolbar integration that shows all your environment variables — grouped, masked, and searchable — directly inside astro dev.
  • Astro Toolbar Routes - A dev toolbar integration that shows a live, clickable route map of your entire project — grouped by type, searchable, and active only in development.
  • Astroscope - A collection of Astro integrations for common server-side needs — logging, tracing, security, i18n, and more.
