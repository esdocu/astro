  • Accessible Astro Launcher - An accessible and powerful launcher (command palette) for Astro websites, built with WCAG compliance and inclusive design principles.

  • Astro Blog TUI Editor - A powerful, Vim-inspired Terminal User Interface (TUI) editor designed specifically for managing and editing Astro blog posts.
  • Astro Snowfall - A lightweight, performant canvas-based snowfall background animation component built with Astro and TypeScript.
  • Astro Swiper.js WebGL - Astro component for WebGL image transitions using displacement maps, powered by CurtainsJS and Swiper.js.
  • Astro Cache - Astro integration providing disk-backed caching utilities with stale-while-revalidate and memoization support.
  • Astro Bun Adapter - An Astro SSR adapter for Bun.serve.

  • Astro Journey - An intuitive timeline component for Astro where entries are spaced based on actual time gaps. Perfect for portfolios, changelogs, and career journeys.
  • Astro Standard.site - Publish your Astro blog to the federated web. This package connects your blog to ATProto (the protocol behind Bluesky) using the Standard.site schema.
  • Statcounter Astro - The official Statcounter integration for Astro.
  • Astro Cloudflare Media - A development-time Astro integration for managing media uploads to Cloudflare (Stream, R2, Images).
  • Frontmatter Core - A build-time tool that scans Astro, Markdown and YAML and extracts a CMS-ready JSON content model from your project.
  • Astropub md - Render any Markdown content in Astro, optionally integrating with any existing configuration.
  • Astro Bottomsheet - Draggable bottom sheet / drawer component for Astro with TypeScript support. Perfect for creating mobile-friendly bottom drawers, side panels, or modal-like interfaces with smooth drag interactions.
  • Astro WebGL Hover - WebGL image hover effects for Astro with displacement transitions using Curtains.js and GSAP.
  • Starwind UI form builder - Build forms visually and export production-ready Astro code.
  • Astro SessionKit - Simple session access and route protection for Astro applications.
  • Astroscope i18n - i18n for Astro + React islands — automatic tree-shaking, parallel loading, any translation source.
  • Astro LLM Translator - An Astro integration that automatically translates your Markdown and MDX content for Astro & Starlight using LLMs.
  • Astro Liquid Glass - A design system for Astro inspired by Apple’s WWDC 2025 Liquid Glass aesthetic. Features glassmorphism effects, motion-responsive highlights, and iOS-style interactive components.
  • LLM Enhancements - An Astro integration that exposes pages as .md endpoints.
  • CribOps Astro - Deploy Astro sites to AWS with CribOps - S3, CloudFront, Lambda SSR, and Hive caching.
  • Astro i18next - An i18next solution for Astro with zero JS output and hot reload support.
  • Astro WebGL Transition Canvas - An Astro “drop-in” component for creating fluid and seamless page transitions using WebGL (Three.js) and Astro’s <ClientRouter />.
  • Vite Plugin Astro Prerender - A Vite plugin for Astro that prerenders components to static HTML and generates optimized CSS. Perfect for lazy-loading below-the-fold content to improve initial page load times.
  • Amplify Astro Adapter - AWS Amplify adapter for Astro with built-in cookie-based sessions.
  • Writenex Astro - Visual editor for Astro content collections - WYSIWYG editing for your Astro site.
  • Astro YM - A Yandex Metrika integration for Astro with full ClientRouter support.
  • Astro TokenKit - A powerful, type-safe API client for Astro with automatic token rotation, session management, and seamless context integration.
  • Shipyard - The complete toolkit for building stunning websites with Astro. A comprehensive collection of Astro integrations
  • Astro Broken Links Checker - An Astro integration that checks for broken links in your website during static build.
  • Astro Build Timestamp - A tiny Astro integration that writes the exact time a site was built into a static file during astro build.
  • Astro Canonical - Build-time canonical URL validation for Astro. Enforces site origin and trailing-slash rules and fails builds on SEO mistakes.
  • Astro Build Info - A minimal Astro integration that generates a machine-readable /build-info.json file at build time with build metadata.
  • Ember Astro - A plugin for astro.build for using Ember in astro components.
  • Wuchale - A framework-agnostic compile-time internationalization toolkit that automatically extracts and replaces translatable messages at build time.
  • Biel.ai Chatbot - An Astro integration for Biel.ai, adding AI-powered search and a documentation chatbot that answers technical questions directly from your docs.
  • Scalar Astro - An Astro integration to interact with the Scalar API.
  • Astro Build ID - A tiny, static-first Astro integration that writes a single build identifier file at build time.
  • Astro Fatsoma - A minimal, SSR-first Astro integration for fetching and normalising public event data from the Fatsoma API.
  • Astro Head Check - A lightweight Astro integration that validates essential HTML <head> metadata at build time.
  • Astro LLM - Deterministic, build-time content extraction for Astro sites, designed for LLMs that generates a single, clean, static context file from your built HTML.
  • Astro Legal - A static‑first Astro integration that scaffolds professional legal policy pages from a single config file.
  • Astro Humans - A static‑first Astro integration that generates a clean, human‑readable humans.txt file at build time.
  • Astro HTTP - Gives you full, explicit control over security, privacy, cache, and legacy HTTP headers using a single, auditable configuration file.
  • Astro No Crawl - Blocks search engines from crawling non-production Astro sites by generating a restrictive robots.txt at build time.
  • Astro No Email - A tiny, static-first Astro integration that protects email addresses from basic scrapers by converting them into HTML entities at build time.
  • Astro noindex - An Astro integration that automatically injects a <meta name="robots" content="noindex, nofollow"> tag into generated HTML for non-production builds.
  • Astro Security - An Astro integration that deterministically generates an RFC 9116–compliant security.txt file at build time.
  • Astro Status - A tiny Astro integration that generates a machine-readable /status.json file at build time for static sites.
  • Astro Strip Whitespace - Strip inter-node whitespace (whitespace-only gaps between elements/expressions) in Astro and Svelte templates.
  • Astro Plugin Logseq - Turn Logseq into a headless CMS for your Astro blog content collections.
  • Gila Astro CSP - Astro 5+ integration for automatic SRI (Subresource Integrity) and CSP (Content-Security-Policy) generation with nginx support.
  • Intellitester Astro - An Astro integration for AutoTester - SSR and hydration testing for Astro applications.
  • 🌟 Starlight Tags - A plugin to add tags to your Starlight docs for content organization and discovery.
  • Astro WP Integration - A WordPress integration for Astro with live loaders, static loaders, client API, and Gutenberg block support.
  • Starwind UI Theme Designer- Customize your Starwind UI theme with real-time preview.
  • Yalidine Astro - An Astro integration for the Yalidine SDK.
  • 🌟 Starlight Telescope - A Starlight plugin to quickly navigate to any page in your Starlight docs with fuzzy search and keyboard-first navigation.
