Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. 🛠

  • Astro Auto Load - Automatic component-level data loading for Astro SSR.
  • Astro Content Viewer - A Dev Toolbar app for Astro that visualizes your content collections, Zod schemas, and associated variables.
  • Astro Dynamic Component - An Astro integration that enables dynamic component imports with glob patterns and client directives support.
  • Astro Shortlinks - Automatically create shortlinks for your Astro site.
  • Astro Tips - Beautiful, customizable tip and notice blocks for Astro with 18 built-in styles and auto dark mode support.
  • Astro Compress - A powerful compression integration for static assets in Astro.
  • 🌟 Starlight GitHub Stats - A Starlight plugin to display GitHub repository statistics (stars, forks, and latest version) in the header of your documentation site.
  • Simple Tips - A simple and easy-to-use Astro integration for creating beautiful tooltips in Markdown and MDX files.
  • Astro Speed Measure - Measure Astro build performance end-to-end. Wraps integrations and Vite plugins, aggregates per-page and asset timings, and emits human, JSON, or HTML reports.
  • EdgeOne Astro Adapter - An adapter that allows you to deploy your Astro site to EdgeOne Pages.
  • Astro Pirsch Proxy - An Astro integration that exposes a Pirsch analytics proxy.
  • Auto Param Astro - Astro integration that adds configured query parameters to external links at build time.
  • Astro SEO - SEO component for Astro with full TypeScript support.
  • Astro Deck - Command deck for Astro - Admin hub with CMS, AI content generator, and developer tools.
  • sweetcorn - JavaScript image dithering tools for Sharp. Turn smooth pixels into crunchy kernels.
  • Astro Content AI Translator - Astro integration for AI-powered content translation using OpenAI.
  • CMS Marker - An Astro integration that automatically marks HTML elements with unique identifiers and generates a manifest mapping content to source files and line numbers. Perfect for building CMS editors that need to track where content originates in your codebase.
  • Astro Click to Source - An Astro integration for Alt+Click to open source files in your editor during development.
  • ez i18n - Cookie-based i18n for Astro + Vue + React. No URL prefixes, reactive language switching.
  • Astro Marp - An Astro integration that transforms .marp Markdown slide sources into optimized presentation pages with full asset pipeline integration and server-side Mermaid diagram rendering.
  • Astro NocoDB - An Astro loader package that integrates NocoDB data into Astro content collections.
  • Spectre UI Astro - First‑class Astro layer for Spectre UI. Typed, SSR‑friendly Astro components (Button, Card, Input and more) that wrap Spectre UI’s classes and tokens.
  • Astro Shield - An Astro integration to enhance your website’s security with SubResource Integrity hashes, Content-Security-Policy headers, and other techniques.
  • Writenex Astro - Visual editor for Astro content collections - WYSIWYG editing for your Astro site.
  • Dynamic Storage Interface - Reusable Astro components for cloud storage file management. Currently implements S3-compatible storage providers (AWS S3, Backblaze B2, MinIO, etc.).
  • Bejamas UI - Astro-Native Building Blocks for Modern Design Systems.
  • Vault CMS - Use Obsidian as a content management system for your Astro website.
  • Pageel - A powerful, client-side Content Management System (CMS) for Astro and Next.js for managing Markdown/MDX content and images directly on GitHub, Gitea, or Gogs repositories.
  • Astroscope - A collection of Astro packages to make production-ready applications: HTTP proxy for Astro SSR, Open Telemetry support, extended CSRF protection, and lifecycle hooks for Astro SSR (startup/shutdown) to run initialization and cleanup code for your Astro server.
  • Photosuite - A simple yet feature-rich image integration that integrates lightbox, EXIF data, path resolution, and more into a single, zero-config package.
  • Astro Spearly - An integration that makes it easy to use the Spearly CMS REST API with Astro.
  • Astro GraphQL - An Astro content loader for GraphQL endpoints. Fetch data from any GraphQL API and use it as Astro content collections.
  • Astro Env Bun - An Astro integration to add .env types defined using envField to Bun’s env.
  • Storybook Astro - Storybook framework for Astro components. Render .astro files directly in Storybook with full support for props, controls, and automatic source code generation.
  • Astro IndexNow - An Astro integration that automatically submits all built pages to IndexNow after each build, helping search engines discover updates faster.
  • Expressive Code Collapsible - A plugin for Expressive Code that adds collapsible functionality to code blocks, automatically collapsing long code with expand/collapse controls.
  • Leaflet Loader Astro - Includes both a build-time and live loader for fetching Leaflet document records from your personal data server to then be used on your Astro site.
  • Astro Grab is a dev-only tool that helps you quickly locate and grab source code from your Astro components. Simply hold Cmd+G (Mac) or Ctrl+G (Windows/Linux) for one second, then click any element to copy its source snippet to your clipboard.
