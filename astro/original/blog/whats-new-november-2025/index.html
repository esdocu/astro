Astro Team News

ViteConf talks now available on YouTube: See all the talks from ViteConf 2025, including “The Future of Astro” by Matt Kane and “Making Meaningful Contributions to Open Source” by Sarah Rainsberger

Speaking: Core maintainer Reuben Tier will be speaking about Astro’s island architecture December 1 at React Advanced London. It’s not too late to sign up to see this virtual talk!

Meetup: The new Astro Japan Community will hold their first in-person meetup on February 4, 2026! Join Astro maintainer and community organizer Kenji Tomita to hear talks from notable community members including Kyosuke Nakamura, who is no stranger to Astro Docs translations!

Live Stream: Hang out on a Discord call with core maintainer Florian Lefebvre for a live, pair-programming refactoring session with “friend of the family” (actual family, for some of us!) J. B. Rainsberger.

You can find these upcoming events on the official Astro Luma event calendar. Have an Astro event to announce? Let us know so we can add it!

Learn in your language

This month, a dedicated group of community volunteers translated the official Astro Docs Tutorial into German! 10 people, 33 files changed, 51 days, 96 commits! Please join us in a hearty “Vielen Dank!” to all participants, and see our i18n contributing docs if you’re interested in translating Astro Docs into your language.

Prefer to learn in Spanish? Don’t miss the new, free Midu.dev course for Astro. Starting from scratch, you’ll finish the course with a personal Astro blog integrated with a headless CMS.

And there’s never been a better time to register for Chris Pennington / Coding In Public’s Learn Astro course - now 33% off for the holidays! Use the affiliate link in this post, or in Astro Docs, and a portion of your purchase is donated to the Astro open-source project!

Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. 🛠

  • Astro Web Vitals - A lightweight Web Vitals monitoring component for Astro with debug overlay and analytics support.
  • Astro Capo - Automatically sort your head with capo.js in Astro.
  • Astro i18nya - I18n support for Astro.
  • Astro AI Coauthor - A developer-first AI documentation assistant for Astro.
  • Astro Cloudflare Pages Static - An Astro integration for client-side dynamic routes with Cloudflare Pages.
  • Astro Directify - Declarative server-side control-flow directives for Astro.
  • Astro WhatsApp - An Astro component to show a WhatsApp fixed floating icon in your website.
  • Mona Kiosk - An Astro integration for monetizing your content with Polar.sh.
  • Zephyr Astro - An Astro integration for Zephyr Cloud deployment and analytics platform.
  • Astro Orbit - A theme package for Astro projects featuring a design system, reusable components, and Tailwind CSS v4 integration.
  • Astro Snapshot - An Astro integration for generating screenshots of your pages automatically at build time.
  • Silk Astro - Astro integration for Silk - zero-runtime CSS-in-TypeScript with islands architecture and partial hydration support.
  • Scalar API Reference for Astro - A Scalar API Reference Component for Astro.
  • EdgeOne Astro - An adapter to deploy your Astro site to EdgeOne Pages.
  • Astro Auth - An Astro integration for Auth.js that provides seamless authentication with multiple providers, session management, and UI primitives that feel natural in Astro.
  • Astro Dynamic Assets - An Astro integration to help you build custom images like Open Graph Image, Instagram post, etc., using React.
  • Astro Trakt.tv Loader - A Trakt.tv content loader for Astro’s content layer.
  • Astro HTML Minifier Next - An integration to automatically minify all HTML assets generated by Astro during the build process using the provided html-minifier-next options.
  • AstroJS Atom - A library for generating Atom RSS feeds in Astro projects.
  • Vitest Browser Astro - Test Astro components in real browsers with Vitest Browser Mode.

… and some built with Astro!

  • Git Views - A GitHub profile and repo view counter you can add to your website.
  • Release Image Generator - Generate random images with customizable patterns, colors, and text overlays.
  • Starlight Detector - Check if websites are built with Starlight or not. Similar, but not as powerful as isAstro.
