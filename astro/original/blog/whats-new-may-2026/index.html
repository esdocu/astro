Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. 🛠

  • Astrobook: UI component playground that supports multiple frameworks, including React, Vue, Preact, Svelte, Solid, Lit, and Astro. It offers a unified environment to develop, test, and showcase components.
  • @hoangyell/gallery: Astro image gallery with lightbox, pyramid multi-row layout, and accessible controls.
  • @brandon_m_behring/book-scaffold-astro: Astro 6 + MDX toolkit for long-form technical books.
  • @scale.digital/astro-bun: A native Bun adapter for Astro 6 — runs SSR, hybrid, and static sites directly on Bun.serve
  • astro-jsonfeed: Generate a JSON Feed for your Astro site.
  • astro-oembed: Embed any URL in your Astro site using the oEmbed protocol.
  • eminence-astro-suite: Astro components and integration for managing page head, metadata, social sharing, icons, and SEO outputs.
  • astro-lang-i18n: 📦️ Smart multi-dimensional routing for Astro i18n projects.
  • astro-nice-asset-filenames: Astro integration for short, readable asset URLs.
  • astro-headers-file: Astro integration that uses your existing configuration to write a static headers file for platforms like Cloudflare Pages, Netlify and Vercel, and others.
  • @dallay/astro-icon: A community-maintained fork of astro-icon by Nate Moore, updated for Astro v6.x and Cloudflare Workers compatibility.
  • @develjet/astro-icon: A lightweight Iconify integration for Astro with support for local SVG icons.
  • @erfianugrah/astro-image-hq: High-quality image service for Astro: encoder routing across avifenc, NVENC, and sharp with content-aware shadow detection.
  • @mannisto/astro-masonry: A masonry layout component for Astro.
  • @mgasme/elu: Composable Astro select component. Single, multi, searchable, grouped. Zero runtime dependencies.
  • @northsoon/phosphor-icons-astro: Phosphor icons as fully-typed Astro components — size, weight, color, mirrored and aria-label props out of the box.
  • @rafters/astro-data: Loader and action contracts for Astro 6 sites.
  • @rgglez/astro-aside: Astro aside/callout component with note, tip, caution, warning, and danger variants. Requires Tailwind CSS.
  • @rgglez/astro-downloadchecklist: Astro component that wraps a markdown task list and adds a CSV download button.
  • astro-bulma: Astro component library wrapping Bulma CSS
  • astro-hotspot: Astro image service: hotspot cropping (x/y or ‘70% 34%’) via Sharp’s extract+resize.
  • astro-recommends: Affiliate link cloaking for Astro. A static-site replacement for ThirstyAffiliates.
  • astro-site-shell: A CLI-style terminal widget for Astro sites. Navigate content through a pseudo shell interface.
  • ezqr-astro: Astro component that embeds the EZQR free QR code generator on your site.

CMS/text editors

  • WebcoreUI CMS: A CMS built for Astro with WebcoreUI. Includes a visual block builder, flexible content modeling, and a DB layer that just works.
  • @vulsecms/core: Astro-native headless CMS for Cloudflare (D1, R2, Workers).
  • astro-blog-kit: A ready-to-use blog system for Astro with WordPress headless support, optional i18n, multiple layouts, and a comment system.

Content loaders

  • notion-loader: Astro Content Loader for Notion as a CMS. Fork of astro-notion/notion-astro-loader with Astro 6 compatibility patches.

Dev toolbar apps

  • astro-agent-annotate: Astro dev toolbar plugin: Alt+Click any element, annotate it, copy notes + source context to clipboard for AI agents.
  • astro-inspect-clip: Astro dev-toolbar app to inspect elements and copy source info + instructions to clipboard.

Astro integrations by and for other tools

  • @imagekit/astro: Astro SDK for ImageKit.io - optimized image & video delivery with transformations
  • @a1local/astro-adpages: Astro integration helpers for AdPages local SEO and landing-page QA.
  • @dualmark/astro: Astro integration for the Dualmark AEO framework.
  • @generative-dom/astro: Official Astro integration for Generative DOM.
  • @hover-dev/astro: Astro integration for Hover. Injects the floating chat widget into your Astro dev site and starts the local agent-driving service.
  • @love-rox/tcy-astro: Astro integration and <Tcy> component for Japanese tate-chu-yoko (縦中横) auto-wrap
  • @pico-intl-dev/astro: Astro integration for pico-intl - SSR-ready, zero-overhead i18n for Astro projects
  • @sudodevstudio/fastly-for-astro: Astro v6 adapter for Fastly Compute. Dynamic SSR, server islands, and KV-backed static assets with no app-code changes.
  • @unirate/astro: Astro integration for UniRate — build-time currency exchange rate snapshots, <Currency /> and <Rate /> components, and runtime conversion helpers.
  • astro-resend-router: Astro integration for using Resend as an email-based mailing list router.
  • astro-typekit: Astro-native font loading for Google Fonts and local font files.
  • astro-x402: Astro middleware integration for the x402 Payment Protocol.

Markdown, Text, and more

  • @philnash/astro-related-content: An Astro integration that generates semantically related content for Markdown and MDX sources.
  • @casoon/astro-site-files: Astro integration that generates robots.txt, llms.txt, sitemap.xml, security.txt, and humans.txt at build time from typed configuration.
  • @jdevalk/astro-markdown-alternate: Astro integration that injects <link rel="alternate" type="text/markdown"> into article pages at build time.
  • astro-mark-don: Astro integration that generates a .md version of every page at build time, optimized for LLM crawlers.
  • astro-md-content-negotiation: Astro integration that generates Markdown versions of every page at build time, enabling HTTP content negotiation.
  • astro-slop: The most complete and flexible Astro integration for LLMs and generating markdown versions of pages. Per-page .md siblings, llms.txt index, content negotiation, and link rel=alternate.
  • @rafters/astro-meta: Build-time artifacts for crawlers and LLMs on Astro 6: JSON-LD, llms.txt, robots, sitemap, OG images, and build-time GEO readability scoring.
