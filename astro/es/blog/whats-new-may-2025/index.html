Tips y Herramientas

Las últimas utilidades e integraciones creadas por la comunidad para ayudarte a construir con Astro. 🛠

  • Astro Prettier Response - Este paquete inyecta un middleware que ejecuta el formateador Prettier para toda la salida, haciéndola sin minificar y bonita.
  • Astro Strapi Blocks - Bloques de Astro para el campo Rich Text Block de Strapi.
  • Piny Tailwind Visual Editor - Una extensión de VSCode que te permite editar visualmente Tailwind CSS directamente en tus archivos de React y Astro.
  • Astro Image Dithering - Aplica dithering a tus imágenes antes de servirlas a tu audiencia para obtener tiempos de carga de imágenes más rápidos combinados con una menor huella de carbono.
  • Remark Auto Diff - Un plugin de remark que proporciona una sintaxis alternativa para crear bloques de código diff en Markdown.
  • Schema-driven Astro Starter Kit - Un kit de inicio listo para producción que aporta datos estructurados, optimización SEO y desarrollo basado en esquemas a tus proyectos de Astro.
  • Astro Loader i18n - Un cargador de contenido glob de Astro para archivos i18n y estructuras de carpetas.
  • WebApp Astro PWA - Una librería de componentes de Astro lista para usar que añade soporte de Progressive Web App (PWA) a tus proyectos de Astro.
  • Astro Simple Modal - Un modal simple y altamente personalizable para Astro — configuración mínima, control total del estilo.
  • Astro Maintenance - Añade fácilmente páginas de mantenimiento y de "próximamente" a tu sitio de Astro.
  • Astro Iubenda - Esta integración de Astro obtiene y proporciona contenido de Política de Privacidad, Política de Cookies y Términos y Condiciones de Iubenda para tu proyecto de Astro.
  • Light-Auth - Un framework de autenticación ligero, seguro y fácil de implementar con un ejemplo de Astro Hello World.
  • Astro Wordpress ACF Generator - Un generador de código PHP para WordPress Advanced Custom Fields (ACF).
  • Astro Daisy Theme Toggle - Componente para añadir un tema oscuro y claro a tu proyecto de AstroJS y DaisyUI.
  • Astro OG Images - Otra integración de Astro para generar imágenes de Open Graph. Más ligera que la mayoría de las alternativas.
  • Astro Starter Kit: Component Package - Una plantilla para una librería de componentes de Astro. Usa esta plantilla para escribir componentes que puedas usar en múltiples proyectos o publicar en NPM.
  • Flotiq Astro SDK - Generador dedicado de colecciones de Astro para Flotiq Headless CMS.
  • Astro Mailer - Una integración de Astro que te permite enviar notificaciones por correo electrónico a través de cualquier servicio SMTP usando Nodemailer.
  • Astro Resend - Una integración de Astro que te permite enviar fácilmente notificaciones por correo electrónico para envíos de formularios usando Resend en un proyecto de Astro.
  • Remark Livecodes for Astro - Un playground de código abierto del lado del cliente con soporte para Astro.
  • 🌟 Starlight Custom Navigation - Mejora tu sitio de documentación con funciones de navegación de página personalizadas, incluyendo soporte para atajos de teclado.
  • 🌟 Starlight Code Block Fullscreen - Añade funcionalidad de pantalla completa a los bloques de Expressive Code en tu documentación.
  • Astro Adapter Nekoweb - Un adaptador para Astro que despliega tu aplicación automáticamente en Nekoweb.
  • Astro PlantUML - Una integración de Astro para renderizar diagramas de PlantUML en tus archivos de markdown.
  • Astro Tips - Bloques personalizables de consejos y avisos para Astro con 18 estilos integrados y soporte automático de modo oscuro.
  • Vemetric Astro SDK - Rastrea tus datos a través del SDK de Astro para Vemetric.
  • Astroboard - Transforma tu CMS de Umbraco con un panel dinámico.
  • Inox Tools Cut-short - Una integración de Astro que te permite detener el procesamiento de una solicitud al instante y devolver una respuesta personalizada, ahora con una opción cancelPrerender para omitir dinámicamente la generación de una página específica. En lugar de renderizar una página 404, esta utilidad asegura que la página se omita completamente de tu salida estática final.

… y algunos construidos con Astro:

  • Astro Life in Weeks - Un proyecto de Astro para crear tu propio “Tu vida en semanas”, una visualización interactiva donde cada rectángulo representa una semana de tu vida.
  • Ubikute Relay - Una nueva página de estado para que comunidades y organizaciones la autohospeden, con el frontend construido en Astro.
  • html2astro - Arrastra y suelta tu HTML. Conviértelo instantáneamente en un proyecto de Astro.
  • Historical Timeline Calculator - Crea y exporta un gráfico de líneas de tiempo históricas.
