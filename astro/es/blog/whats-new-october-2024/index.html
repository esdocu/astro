Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro.

  • astro-react-i18next - Integra i18next y react-i18next de forma transparente en tu sitio web de Astro para proporcionar un soporte robusto de i18n para componentes de React.
  • Astro Resend - Envía fácilmente notificaciones por email para envíos de formularios usando Resend en un proyecto de Astro.
  • DeWP - Usa los datos de tu WordPress en proyectos de Astro
  • starlight-site-graph - Un plugin de Starlight que añade un componente de grafo a la barra lateral derecha del sitio.
  • Astro NPM Loader - Carga información sobre paquetes de NPM en una colección de contenido.
  • Local AstroDB Viewer - Visualiza e interactúa con archivos .db directamente en Visual Studio Code.
  • Astro Mail Obfuscation - Protege direcciones de email y números de teléfono en el código fuente de tu app de Astro de los bots usando ofuscación basada en XOR.
  • react-alpine-store - Permite la sincronización entre estado reactivo a través de React y el $store de Alpine. Mira más información sobre el $store de Alpine aquí.
  • Expressive Code Color Chips - Un plugin para añadir vistas previas de color a tus ejemplos de código CSS
  • Astro AO3 loader - Un cargador de datos de AO3 para Astro usando la capa de contenido experimental y AO3.js para cargar datos de AO3 a tu sitio web de Astro.
  • Portal Gun - Portales del lado del servidor para añadir elementos HTML fuera del componente que los define.
