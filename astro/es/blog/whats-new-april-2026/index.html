Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • @fujocoded/astro-authproto: Añade autenticación ATproto a tu sitio de Astro. Permite a tus visitantes iniciar sesión en tu sitio web usando su cuenta en cualquier servicio ATproto (como Bluesky).
  • @jdevalk/astro-seo-graph: Integración de Astro con componente SEO, fábricas de rutas, agregador de colecciones de contenido, helpers de contenido Zod.
  • @markdy/astro: Componente island de Astro para escribir animaciones MarkdyScript y generar escenas animadas directamente en el navegador.
  • @flagify/astro: Una integración de Astro para evaluar feature flags en páginas y componentes, con una dev toolbar para overrides.
  • bodegacat: Integración de Astro para tiendas e-commerce de Bodega Cat (Cloudflare + Stripe).
  • astro-woocommerce: Integración type-safe de WooCommerce para Astro. Obtén productos, categorías y gestiona el estado del carrito con soporte completo de TypeScript.
  • @hedystia/astro: Usa componentes @hedystia/view dentro de Astro.
  • @zdenekkurecka/astro-consent: Integración de Astro para consentimiento de cookies compatible con GDPR/ePrivacy: banner, modal de preferencias, API en runtime, estado de consentimiento por categorías, seguro con strict-CSP, funciona con o sin View Transitions.
  • astro-magic-move: Transformación animada de código para Astro, impulsada por Shiki Magic Move. Tokenización en build-time, JS de cliente sin framework, theming con variables CSS.
  • @ahmttyydn/astro: Componente de referencia de API Scalar para Astro. Renderiza una hermosa referencia de API basada en un documento OpenAPI/Swagger.
  • astro-pigment: Un tema de documentación para Astro 6 con modo oscuro, playgrounds interactivos y endpoints SEO.
  • @casoon/astro-sitemap: Integración de Astro que genera sitemap.xml en build-time — soporta fuentes dinámicas, reglas basadas en patrones, hreflang i18n, hook de serialización y división de sitemap-index.
  • @twodft/astro-icon: Un fork mantenido por la comunidad de astro-icon por Nate Moore, actualizado para Astro v6.x y compatibilidad con Cloudflare Workers.
  • @omniaura/astro-grab: Mantén una tecla, haz hover, clic — captura el contexto fuente de cualquier elemento para agentes de IA. Devtool de Astro.
  • @casoon/astro-crawler-policy: Control de crawlers basado en políticas para Astro — genera robots.txt y llms.txt con presets, reglas por bot, registro de crawlers de IA y auditorías en build-time.
  • astro-bangumi: Una integración de Bangumi para Astro, similar a hexo-bilibili-bangumi.
  • astro-include-asset: Incluye assets en tu página escribiendo una directiva remark personalizada en tu archivo Markdown/MDX.
  • @pliosoft/sites-sdk: Componentes, layouts y design tokens compartidos de Astro para sitios de clientes alojados por Pliosoft.
  • astro-cloudflare-analytics: Un componente de Astro para manejar la instalación manual de Cloudflare Analytics.
  • astro-research-publications: Integración de Astro que sincroniza tus publicaciones académicas de Google Scholar, Semantic Scholar, OpenAlex y/o ORCID y renderiza una página de publicaciones.
  • astro-llms-md: Genera llms.txt, llms-full.txt y archivos markdown individuales desde tu sitio de Astro con esta integración de Astro.
  • @red-glare/astro: Integración de Astro para construir sitios de documentación federal con estilo USWDS usando Comark.
  • @soufianeh/annotate-code: Vincula líneas de código con explicaciones — una integración de Astro y plugin remark con resaltado al hacer hover.
  • @jessgaspardev/astro-seo: Un componente SEO completo para Astro con Open Graph, Twitter Cards, JSON-LD y más.
  • @dotslide/framework: Un framework moderno de presentaciones para Astro. Construye decks rápidos y portables desde componentes semánticos listos para usar en lugar de código de slides personalizado.
  • astro-omni-compress: Una alternativa drop-in a sharp para el componente <Image> de Astro que funciona donde los binarios nativos de sharp fallan — imágenes Docker alpine, funciones serverless, Vercel Edge, Cloudflare Workers.
  • @at-flux/astro-feature-flags: Feature flags tipados de Astro desde configuración JSON. Incluye renderizado específico por entorno y poda desde elementos hasta rutas completas.
  • @twodft/astro-favicons: Un generador todo-en-uno de favicons y assets PWA para proyectos de Astro. Automatiza la creación de favicons, manifest y soporta hot reloading para un desarrollo eficiente.
  • @tsparticles/astro: Crea fácilmente animaciones de partículas, confeti y fuegos artificiales altamente personalizables con componentes listos para usar y úsalos como fondos animados.
  • @obaronai/astro-ai-readiness: Toolkit de AI Readiness para Astro.
  • astro-tinylytics: Componentes de Astro para analíticas de Tinylytics, botones de kudos y webmentions.
  • astro-meta-engine: Motor potente y type-safe de metadatos y JSON-LD para Astro.
  • astro-gtm-lite: Google Tag Manager ligero compatible con Astro View Transitions (client router).
  • @mintlify/astro - Sincroniza contenido de docs estilo Mintlify en colecciones de contenido de Astro y proporciona componentes MDX para renderizar páginas de docs.
  • astro-dgmo: Integración de Astro para renderizar diagramas DGMO desde bloques de código cercados en build-time.
  • astro-opengraph - Una integración para crear y gestionar tus imágenes de Open Graph, incluyendo una app de dev toolbar para previsualizar la imagen y las og meta tags en la página.
  • @ayco/astro-resume: Utilidades para serializar datos del servidor para usarlos en el cliente.
  • @better-translate/astro: Un toolkit i18n type-safe para TypeScript.
  • astlide: Un framework de presentaciones de slides basado en Astro — como Slidev, pero para el ecosistema de Astro.
  • astro-feedkit: Una integración de Astro para feeds RSS, Atom y JSON de contenido completo.
  • astro-mdx: Una integración de Astro para mapeo de directivas MDX a componentes, overrides de elementos y auto-imports.
  • astro-search-plugin: Búsqueda agnóstica al framework y type-safe para Astro 5+ impulsada por Orama.
  • astro-agent-optimized: Salida optimizada para agentes para Astro. Inspirado en laravel/pao.

Plugins

  • Astro Cursor Plugin: Plugin de Cursor para Astro que incluye el MCP oficial de Astro Docs, reglas con mejores prácticas y estándares de código, y varias skills como añadir integración, colección de contenido, crear componente, búsqueda en docs y migrar.
  • ESLint astro/no-prerender-export-outside-pages: Una regla de eslint-plugin-astro para Astro que advierte cuando se usa export const prerender fuera del directorio pages/.

Skills

  • astro-agent-skills: Una colección curada de skills de agentes que dan a los asistentes de IA conocimiento profundo de los patrones, mejores prácticas y flujos de trabajo de Astro. Funciona con Codex, Cursor, Claude Code y otros agentes que soportan el ecosistema abierto de skills.

CMS/editores de texto

  • star-lite-docs: Tema de documentación estilo Starlight para EmDash CMS — integración de Astro + plugin de emdash en un solo paquete.
  • phage-cms: Integración de Astro: Phage CMS (Cloudflare KV/R2, editor de bloques, rutas de API).
  • nebula-cms: Un sistema de gestión de contenido optimizado y basado en Git, construido específicamente para Astro.
  • @a83/orbiter-integration: Integración de Astro para Orbiter CMS — UI de admin, módulos virtuales, rutas inyectadas.
  • clay-cms: Un backend de contenido nativo de Astro, basado en adaptadores, con un archivo de configuración estilo Payload y una UI de admin construida únicamente con Astro.

Content loaders

  • notro-loader: Un content loader para Astro que usa la API pública de Notion.
  • @dylmye/atproto-standard-site-astro-loader: Loader de Astro para documentos de Standard Site (pckt.blog, leaflet.pub, etc.) - ¡añade tus publicaciones de blog a tu sitio de Astro!
  • @fujocoded/astro-atproto-loader: Loaders de Astro (en vivo y estáticos) para leer registros AtProto en colecciones de contenido de Astro.
  • astro-loader-s3: Un content loader de Astro para cargar archivos Markdown desde almacenamiento de objetos compatible con S3 (AWS S3, Cloudflare R2, DigitalOcean Spaces, y otros).

Apps de dev toolbar

  • astro-click-to-ai: Una app de Dev Toolbar de Astro que captura los elementos clickeados como una lista JSON con anotaciones para asistentes de código de IA.

Starlight

  • 🌟 starlight-theme-terminal: Tema de Astro Starlight con elementos retro y modernos.
  • 🌟 starlight-skills: Un plugin de Starlight que convierte tu documentación en skills instalables por IA.
  • 🌟 starlight-llm-actions: Un plugin de Starlight que añade un menú desplegable de Page Actions a cada página de docs con copiar/ver como Markdown, guardar como PDF, abrir con <provider>
  • 🌟 starlight-recipes: Un plugin de Starlight para crear un sitio web de recetas: modo cocina, diseño imprimible, SEO optimizado a través de Structured Data (JSON-LD) y sistema de valoraciones.
