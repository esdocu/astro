Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro.

  • Astrobook - Un playground minimalista de componentes UI que soporta múltiples frameworks incluyendo React, Vue, Preact, Svelte, Solid, Lit y Astro. Ofrece un entorno unificado para desarrollar, probar y mostrar componentes.
  • create-krew - Una herramienta CLI para configurar rápidamente un proyecto de Astro con repositorio de GitHub, despliegue en Vercel y pipeline de CI/CD.
  • Bag of Tricks: <InspectionChamber /> - Añade un panel de control que puedes usar para examinar tus animaciones de view transition.
  • stripe-astro-loader - Obtén datos de la API de Stripe y úsalos en colecciones de Astro.
  • Loader de contenido de Chuck Norris facts - ¡Sabías que alguien iba a hacerlo!
  • @jcayzac/astro-build-cache - Una caché de compilación simple para artefactos que puedes estar generando durante una compilación de Astro.
  • @jcayzac/astro-classlist - Una pequeña utilidad para forzar/limpiar class:list cuando quieres pasarla a un componente hijo.
  • notion-astro-loader - Carga páginas de una base de datos de Notion y luego renderízalas como páginas en una colección.
  • Kit de inicio de DatoCMS para Astro
  • astro-loader-gather - Un loader para la Content Layer de Astro similar al loader glob integrado con algunas extensiones para extraer metadatos de las rutas del sistema de archivos.
  • mdxtree - Una herramienta de línea de comandos en Rust que genera una estructura de árbol de archivos en formato MDX para usar con los componentes de Starlight de Astro.
  • astro-social-share - ¡v2.0! Botones para compartir en redes sociales para tu sitio de Astro.
  • astro-dump - Una mejor experiencia de depuración.
  • astro-console-cleaner - Una integración de Astro que elimina las sentencias console.log, console.warn y console.error de tu código durante el proceso de compilación.
  • Astro Breadcrumbs - Un componente configurable de breadcrumbs para Astro.
  • Astroboard - Astroboard es un panel de control de Umbraco para obtener información sobre tus contenidos, recursos y miembros.
  • Inox Tools: Request State - Comparte el estado de la petición entre componentes y del servidor al cliente.
  • Inox Tools: Request Scoped Nano Stores - Permite que Nano Stores se usen en el servidor y conserve su valor del servidor en el cliente.
  • Inox Tools: Content Git Authors - Recupera los autores y coautores de Git involucrados en la escritura de entradas de colecciones de contenido.
