Actualizaciones desde Mission Control

  • 💼 Astro Jobs abre sus puertas.
  • 🦙 TinaCMS usa Astro por defecto, respaldado por el uso y la demanda de la comunidad.
  • 🖼 ImageKit lanza una integración de Astro para redimensionamiento de imágenes y video en tiempo real, optimización automática y subida de archivos.
  • 🚀 Vista previa alfa de Astro 7 continúa. Adelántate con el soporte de Vite 8, el compilador estable de Rust y más.

Últimos lanzamientos

Los lanzamientos menores introducen nuevas características y mejoras. ¡Mira qué hay de nuevo!

  • Astro 6.3 - Enrutamiento avanzado experimental con soporte para Hono, manejo de redirecciones de imágenes e hidratación resiliente de islands.
  • Astro 6.4 - Una nueva canalización de Markdown conectable, un nuevo procesador de Markdown basado en Rust y nuevos helpers de Cloudflare para enrutamiento avanzado.
  • Starlight 0.39 - Barras laterales autogeneradas más flexibles, estilos mejorados y soporte más robusto para documentación multilingüe.

Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • Astrobook: Playground de componentes de UI que soporta múltiples frameworks, incluyendo React, Vue, Preact, Svelte, Solid, Lit y Astro. Ofrece un entorno unificado para desarrollar, probar y mostrar componentes.
  • @hoangyell/gallery: Galería de imágenes para Astro con lightbox, diseño de filas múltiples en pirámide y controles accesibles.
  • @brandon_m_behring/book-scaffold-astro: Kit de herramientas de Astro 6 + MDX para libros técnicos extensos.
  • @scale.digital/astro-bun: Un adaptador nativo de Bun para Astro 6 — ejecuta sitios SSR, híbridos y estáticos directamente en Bun.serve
  • astro-jsonfeed: Genera un JSON Feed para tu sitio de Astro.
  • astro-oembed: Incrusta cualquier URL en tu sitio de Astro usando el protocolo oEmbed.
  • eminence-astro-suite: Componentes e integración de Astro para gestionar el head de la página, metadatos, compartir en redes sociales, iconos y resultados de SEO.
  • astro-lang-i18n: 📦️ Enrutamiento multidimensional inteligente para proyectos i18n de Astro.
  • astro-nice-asset-filenames: Integración de Astro para URLs de assets cortas y legibles.
  • astro-headers-file: Integración de Astro que usa tu configuración existente para escribir un archivo de headers estático para plataformas como Cloudflare Pages, Netlify y Vercel, entre otras.
  • @dallay/astro-icon: Un fork mantenido por la comunidad de astro-icon por Nate Moore, actualizado para Astro v6.x y compatibilidad con Cloudflare Workers.
  • @develjet/astro-icon: Una integración ligera de Iconify para Astro con soporte para iconos SVG locales.
  • @erfianugrah/astro-image-hq: Servicio de imágenes de alta calidad para Astro: enrutamiento de codificadores entre avifenc, NVENC y sharp con detección de sombras consciente del contenido.
  • @mannisto/astro-masonry: Un componente de diseño masonry para Astro.
  • @mgasme/elu: Componente de selección componible para Astro. Simple, múltiple, buscable, agrupado. Cero dependencias en tiempo de ejecución.
  • @northsoon/phosphor-icons-astro: Iconos de Phosphor como componentes de Astro totalmente tipados — props de tamaño, peso, color, espejo y aria-label listos para usar.
  • @rafters/astro-data: Contratos de loader y action para sitios de Astro 6.
  • @rgglez/astro-aside: Componente de aside/callout para Astro con variantes de nota, consejo, precaución, advertencia y peligro. Requiere Tailwind CSS.
  • @rgglez/astro-downloadchecklist: Componente de Astro que envuelve una lista de tareas de markdown y añade un botón de descarga CSV.
  • astro-bulma: Librería de componentes de Astro que envuelve Bulma CSS
  • astro-hotspot: Servicio de imágenes para Astro: recorte por hotspot (x/y o ‘70% 34%’) mediante extract+resize de Sharp.
  • astro-recommends: Ocultamiento de enlaces de afiliados para Astro. Un reemplazo de sitios estáticos para ThirstyAffiliates.
  • astro-site-shell: Un widget de terminal estilo CLI para sitios de Astro. Navega por el contenido a través de una interfaz de pseudo shell.
  • ezqr-astro: Componente de Astro que incrusta el generador gratuito de códigos QR de EZQR en tu sitio.

CMS/editores de texto

  • WebcoreUI CMS: Un CMS construido para Astro con WebcoreUI. Incluye un constructor visual de bloques, modelado flexible de contenido y una capa de base de datos que simplemente funciona.
  • @vulsecms/core: CMS headless nativo de Astro para Cloudflare (D1, R2, Workers).
  • astro-blog-kit: Un sistema de blog listo para usar para Astro con soporte headless de WordPress, i18n opcional, múltiples diseños y un sistema de comentarios.

Content loaders

  • notion-loader: Astro Content Loader para usar Notion como CMS. Fork de astro-notion/notion-astro-loader con parches de compatibilidad para Astro 6.

Apps de dev toolbar

  • astro-agent-annotate: Plugin de la barra de herramientas de desarrollo de Astro: Alt+Clic en cualquier elemento, anótalo, copia notas + contexto del código fuente al portapapeles para agentes de IA.
  • astro-inspect-clip: App de la barra de herramientas de desarrollo de Astro para inspeccionar elementos y copiar información del código fuente + instrucciones al portapapeles.

Integraciones de Astro hechas por y para otras herramientas

  • @imagekit/astro: SDK de Astro para ImageKit.io - entrega optimizada de imágenes y video con transformaciones
  • @a1local/astro-adpages: Helpers de integración de Astro para SEO local y QA de landing pages de AdPages.
  • @dualmark/astro: Integración de Astro para el framework AEO de Dualmark.
  • @generative-dom/astro: Integración oficial de Astro para Generative DOM.
  • @hover-dev/astro: Integración de Astro para Hover. Inyecta el widget de chat flotante en tu sitio de desarrollo de Astro e inicia el servicio local de conducción de agentes.
  • @love-rox/tcy-astro: Integración de Astro y componente <Tcy> para auto-wrap japonés tate-chu-yoko (縦中横)
  • @pico-intl-dev/astro: Integración de Astro para pico-intl - i18n listo para SSR con sobrecarga cero para proyectos de Astro
  • @sudodevstudio/fastly-for-astro: Adaptador de Astro v6 para Fastly Compute. SSR dinámico, server islands y assets estáticos respaldados por KV sin cambios en el código de la app.
  • @unirate/astro: Integración de Astro para UniRate — instantáneas de tasas de cambio de moneda en tiempo de build, componentes <Currency /> y <Rate />, y helpers de conversión en tiempo de ejecución.
  • astro-resend-router: Integración de Astro para usar Resend como un enrutador de listas de correo basado en email.
  • astro-typekit: Carga de fuentes nativa de Astro para Google Fonts y archivos de fuentes locales.
  • astro-x402: Integración de middleware de Astro para el Protocolo de Pago x402.

Markdown, Texto y más

  • @philnash/astro-related-content: Una integración de Astro que genera contenido semánticamente relacionado para fuentes de Markdown y MDX.
  • @casoon/astro-site-files: Integración de Astro que genera robots.txt, llms.txt, sitemap.xml, security.txt y humans.txt en tiempo de build a partir de una configuración tipada.
  • @jdevalk/astro-markdown-alternate: Integración de Astro que inyecta <link rel="alternate" type="text/markdown"> en las páginas de artículos en tiempo de build.
  • astro-mark-don: Integración de Astro que genera una versión .md de cada página en tiempo de build, optimizada para crawlers de LLMs.
  • astro-md-content-negotiation: Integración de Astro que genera versiones en Markdown de cada página en tiempo de build, habilitando la negociación de contenido HTTP.
  • astro-slop: La integración de Astro más completa y flexible para LLMs y generación de versiones en markdown de las páginas. Archivos .md por página, índice llms.txt, negociación de contenido y link rel=alternate.
  • @rafters/astro-meta: Artefactos en tiempo de build para crawlers y LLMs en Astro 6: JSON-LD, llms.txt, robots, sitemap, imágenes OG y puntuación de legibilidad GEO en tiempo de build.
