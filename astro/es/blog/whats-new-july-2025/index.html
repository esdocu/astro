Últimos lanzamientos

Los lanzamientos menores introducen nuevas características y mejoras. ¡Mira qué hay de nuevo!

  • Astro 5.11 - nuevas funciones de Content Security Policy (CSP), la capacidad de excluirse del streaming de HTML al usar el adaptador de Node.js, y más!
  • Astro 5.12 - soporte TOML para content loaders, valores de entorno raw experimentales, una experiencia de desarrollo mejorada del adaptador de Netlify, y más!
  • Starlight 0.35 - Nuevas formas de personalizar tus docs, incluyendo iconos en asides, atributos para enlaces autogenerados de la barra lateral, procesamiento de slugs de página, y más!

Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • Astro YouTube Loader - Una integración que te permite cargar videos de YouTube usando la YouTube Data API v3, y usar los datos en tu sitio de Astro. Incluye carga de videos de YouTube en runtime experimental para colecciones de contenido en vivo.
  • Resend Newsletter Starter - ¡Crea tu propio formulario de newsletter con Astro, Resend y React Email!
  • Astro SSR Table - Una integración para crear tablas de datos potentes renderizadas en el servidor. Incluye helpers y componentes para paginación del lado del servidor, ordenamiento y búsqueda de texto completo.
  • Continuous Translation - Una GitHub Action para traducir y mantener automáticamente tu documentación de Markdown usando IA con soporte integrado para Astro Starlight.
  • Blogue - Una herramienta CLI para crear y gestionar publicaciones de blog en Markdown con integración de Astro y automatización de flujos de trabajo de GitHub.
  • Astro Leaflet - Un componente nativo de Astro para leaflet, una librería de JavaScript para mostrar mapas interactivos adaptados a móviles.
  • Astro Headstart - Componentes y utilidades de Astro para gestionar el head de un documento HTML.
  • Astro HTML Helper - Una herramienta CLI para transformar HTML plano en sintaxis más idiomática de Astro.
  • Astro Google Tag Manager - Un plugin para integrar Google Tag Manager en tu proyecto, permitiéndote gestionar etiquetas de marketing y analítica directamente desde tu sitio web de Astro.
  • Astro Auto Alias - Genera automáticamente alias de rutas basados en la estructura de tu proyecto.
  • Astro i18next - Integración de i18next para Astro con carga dinámica de namespaces y soporte para TanStack Router.
  • Componente WorldMap para Astro - Un componente de mapa mundial interactivo para Astro con funcionalidad de búsqueda, vistas de datos configurables y contenido modal personalizable.
  • Astro Nanostores i18n - Una versión para Astro de la librería JS NanoStore i18n para hacer tu aplicación web traducible.
  • Astro LLMS text - Una integración que genera un llms.txt basado en tus páginas cuando construyes tu proyecto de Astro.
  • Zastro Web Sockets - Soporte universal de WebSocket para apps SSR de Astro con adaptadores pre-parchados.
  • Expressive Code Fullscreen - Un plugin de Expressive Code que añade un modo de visualización a pantalla completa a tus bloques de código.
  • Astro Routify - Un router de API para Astro, impulsado por un matcher basado en Trie.
  • Astrojs Mixpanel - Una integración de Mixpanel para Astro.js con configuración mínima requerida.
  • Astro m3e - Una librería moderna de componentes UI para Astro basada en el sistema de diseño Material 3 Expressive.
  • Astro React i18next - Integra i18next y react-i18next en tu sitio web de Astro para proporcionar soporte robusto de i18n para componentes de React.
  • Astro WebAudioStream - Integración de Astro para streaming de Web Audio seguro en iOS Safari con reproducción instantánea, carga progresiva y gestión de memoria.
  • Astro ETag - Una integración de Astro para generar archivos sidecar ETag para tu build.
  • Astro Normalize Trailing Slash - Una utilidad simple para normalizar las barras diagonales finales en URLs para coincidir con el trailingSlash de tu proyecto de Astro.
  • Astro Varlock - Una integración de Astro para Varlock para gestionar tus variables de entorno con seguridad de tipos y seguridad.
  • Astro Posthog - Una integración de Astro para PostHog, tracking compatible con GDPR en tu app/sitio de Astro.
  • BCMS Astro Components - Un conjunto de componentes de Astro para usar contenido de BCMS.
  • Astro Obfuscator - Una integración de Astro que te permite ofuscar archivos JavaScript del cliente usando javascript-obfuscator.
  • Astro Get Remote Image - Una integración de Astro que descarga automáticamente imágenes remotas de tu CMS durante el proceso de build y actualiza las referencias HTML para usar rutas locales.
  • Astro Client Pragma - Una integración de Astro que añade automáticamente la directiva client:* adecuada a cada plantilla leyendo un pragma "client:*"; al inicio de cada componente de framework.
  • Astro Web Vitals - Una integración de Astro que habilita el seguimiento del rendimiento real del sitio web y almacena los datos en Astro DB.
  • 🌟 Starlight Auto Drafts - Plugin de Starlight para ajustar el comportamiento por defecto de las páginas borrador y eliminar automáticamente los enlaces de la barra lateral a páginas borrador en modo producción.
