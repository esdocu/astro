Aprende en tu idioma

Este mes, un grupo dedicado de voluntarios de la comunidad tradujo el Tutorial oficial de Docs de Astro al alemán! 10 personas, 33 archivos modificados, 51 días, 96 commits! Únete a nosotros en un caluroso “Vielen Dank!” a todos los participantes, y consulta nuestra documentación de contribución i18n si estás interesado en traducir Docs de Astro a tu idioma.

¿Prefieres aprender en español? No te pierdas el nuevo y gratuito curso de Midu.dev para Astro. Empezando desde cero, terminarás el curso con un blog personal de Astro integrado con un CMS headless.

Y nunca ha habido un mejor momento para registrarte en el curso Learn Astro de Chris Pennington / Coding In Public - ¡ahora con 33% de descuento por las fiestas! Usa el enlace de afiliado en este post, o en Docs de Astro, ¡y una parte de tu compra se dona al proyecto de código abierto de Astro!

Contenido

Esto es lo que llamó nuestra atención este mes:

  • Gestiona tu contenido en MacOS con Astro Editor hecho para las colecciones de contenido de Astro. Obtén validación, referencias entre colecciones, campos de imagen con vistas previas, y componentes MDX multi-framework — todo en una hermosa interfaz inspirada en iA Writer.
  • Gestiona tu contenido en el móvil: mira qué pasó cuando Jonathan Yeong fue a buscar un CMS headless compatible con móvil para Astro.
  • Vende tu contenido con OpenCourse: una plataforma ligera y de código abierto que permite a los creadores de contenido construir, alojar y vender un único curso. Proporciona una alternativa mínima y autoalojada a plataformas como Teachable — ¡construida con Astro!

Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • Astro Web Vitals - Un componente ligero de monitoreo de Web Vitals para Astro con overlay de depuración y soporte de analíticas.
  • Astro Capo - Ordena automáticamente tu head con capo.js en Astro.
  • Astro i18nya - Soporte de i18n para Astro.
  • Astro AI Coauthor - Un asistente de documentación con IA orientado al desarrollador para Astro.
  • Astro Cloudflare Pages Static - Una integración de Astro para rutas dinámicas del lado del cliente con Cloudflare Pages.
  • Astro Directify - Directivas declarativas de control de flujo del lado del servidor para Astro.
  • Astro WhatsApp - Un componente de Astro para mostrar un icono flotante fijo de WhatsApp en tu sitio web.
  • Mona Kiosk - Una integración de Astro para monetizar tu contenido con Polar.sh.
  • Zephyr Astro - Una integración de Astro para el despliegue y la plataforma de analíticas de Zephyr Cloud.
  • Astro Orbit - Un paquete de temas para proyectos de Astro que incluye un sistema de diseño, componentes reutilizables, e integración con Tailwind CSS v4.
  • Astro Snapshot - Una integración de Astro para generar capturas de pantalla de tus páginas automáticamente en el momento del build.
  • Silk Astro - Integración de Astro para Silk - CSS-in-TypeScript sin runtime con arquitectura de islas y soporte de hidratación parcial.
  • Scalar API Reference para Astro - Un Componente de Referencia de API Scalar para Astro.
  • EdgeOne Astro - Un adaptador para desplegar tu sitio de Astro en EdgeOne Pages.
  • Astro Auth - Una integración de Astro para Auth.js que proporciona autenticación fluida con múltiples proveedores, gestión de sesiones, y primitivas de UI que se sienten naturales en Astro.
  • Astro Dynamic Assets - Una integración de Astro para ayudarte a construir imágenes personalizadas como Open Graph Image, publicaciones de Instagram, etc., usando React.
  • Astro Trakt.tv Loader - Un cargador de contenido de Trakt.tv para la capa de contenido de Astro.
  • Astro HTML Minifier Next - Una integración para minificar automáticamente todos los assets de HTML generados por Astro durante el proceso de build usando las opciones proporcionadas por html-minifier-next.
  • AstroJS Atom - Una librería para generar feeds RSS Atom en proyectos de Astro.
  • Vitest Browser Astro - Prueba componentes de Astro en navegadores reales con Vitest Browser Mode.

… ¡y algunos construidos con Astro!

  • Git Views - Un contador de vistas de perfil y repo de GitHub que puedes añadir a tu sitio web.
  • Release Image Generator - Genera imágenes aleatorias con patrones, colores y superposiciones de texto personalizables.
  • Starlight Detector - Comprueba si los sitios están construidos con Starlight o no. Similar, pero no tan potente como isAstro.
