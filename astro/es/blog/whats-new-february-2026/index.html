Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • AIS Logos Astro - Una integración de Astro para cargar logos de organizaciones de AI Safety.
  • Subfont - Una integración de Astro que divide tus archivos de fuentes para que obtengas tamaños de fuente significativamente más pequeños y velocidades de carga más rápidas.
  • Astro Particle Image Viewer - Un componente de Astro que muestra imágenes con un impresionante efecto de animación de partículas WebGL usando Three.js.
  • Astro Openring - Un cargador de contenido personalizado de Astro que combina múltiples feeds RSS/Atom en una sola colección. Imita el comportamiento de openring / openring-rs, una herramienta para generar webrings.
  • Astro Bun Adapter - Adaptador de Astro para Bun con servicio de archivos estáticos optimizado e ISR (Regeneración Estática Incremental).
  • Astro Cache - Integración de Astro que proporciona utilidades de caché SWR (stale-while-revalidate) respaldada en disco y memoización, impulsada por flat-cache y stale-while-revalidate-cache.
  • Astro Skills - Carga y sirve Agent Skills desde tu sitio de Astro a través de URIs well-known.
  • 🌟 Starlight Announcement - Un plugin para añadir anuncios personalizados a tus docs de Starlight.
  • 🌟 Starlight Telescope - Navega rápidamente a cualquier página en tus docs de Starlight con búsqueda difusa y navegación centrada en el teclado.
  • Astroscope Health - Integración de sondas de salud estilo Kubernetes para Astro.
  • Astro Clarity - Integración de Astro para Microsoft Clarity.
  • Entrolytics Astro SDK - SDK oficial de Astro para Entrolytics - analíticas de crecimiento first-party para el edge.
  • Astro Maplibre-YAML - Componentes de Astro para crear mapas interactivos y experiencias de scrollytelling usando configuración YAML.
  • Astro ONREZA Adapter - Adaptador para desplegar aplicaciones de Astro en la plataforma ONREZA con soporte para SSR, modo híbrido, prerenderizado e ISR.
  • VocaSync Astro - Convierte tus publicaciones de blog de Astro en audio narrado con sincronización a nivel de palabra.
  • Astro Font Loader - Carga fuentes en los artefactos de build de Astro desde paquetes de fuentes personalizados.
  • Astro ID Minifier - Un plugin de Astro para minificar los atributos de datos personalizados de Astro y reducir el tamaño del build.
  • Astro Relative URLs - Esta integración de Astro transforma las URLs de recursos y páginas de absolutas a relativas, haciendo que puedas ver el sitio web construido simplemente abriendo su index.html en tu navegador.
  • Astro Routify - Un router de API de alto rendimiento para Astro usando un matcher basado en Trie.
  • Astro XMDX - Una integración de Astro para xmdx, un compilador de MDX de alto rendimiento.
  • Astro Simple Critical CSS - Una integración de Astro que genera CSS crítico usando beasties y lo incrusta en el HTML.
  • Astro Contrast - Analizador de contraste de color WCAG para componentes de Astro. Revisa tus archivos .astro en busca de problemas de contraste de color en tiempo de build, tiempo de desarrollo, o bajo demanda vía CLI.
  • Astro Junction - Integración de Astro v5+ para Junction. Soporte de View Transitions de primera clase.
  • Astro Primitives - Primitivas de Astro inspiradas en Radix UI Themes.
  • Playform Favicon - Una integración de Astro que trae utilidades de generación de favicons a tu proyecto de Astro.
  • CTRK Astro - Una integración de Astro para el parser de telemetría Yamaha Y-Trac CTRK con componentes y composables de Vue.js.
  • Astro Markdown Runtime - Renderiza Markdown en tiempo de ejecución en Astro usando un pipeline AST personalizado que convierte directivas de Markdown en componentes reales de Astro.
  • Zitadel Astro Auth - Una integración completa de Auth.js para aplicaciones de Astro con soporte de TypeScript, adaptadores HTTP agnósticos al framework, y control de acceso basado en roles.
  • Astro Animations - Animaciones ligeras, sin dependencias, activadas por scroll y efectos CSS vinculados al scroll para Astro.
  • Astro Linter - Integración de Astro que añade ESLint con reglas preconfiguradas.
  • Pagefind Proxima - Una integración de Astro para Pagefind con un componente de Búsqueda listo para usar.
  • Agent CMS - CMS headless primero para agentes de IA para Astro 6 + Cloudflare.
  • Astroscope Wormhole - Comparte datos dinámicos del servidor con islas de React y scripts de cliente — tipado, reactivo.
  • Frontman Astro - Integración de Astro para Frontman — herramientas de desarrollo potenciadas por IA que te permiten editar tu frontend desde el navegador.
  • Astro Sveltia CMS - Integración de Sveltia CMS para Astro.
  • Astro Metadata - Componentes de Astro para gestionar el head de tu página — metadatos, compartir en redes sociales, favicons y SEO.
  • Astro UI - Un sistema de UI basado en tokens y clases para Astro.
  • AstroResponsive Privacy - Integración de Astro para Responsive Privacy — protección de PII en tiempo de build para sitios estáticos. Filtra campos de contenido en tiempo de build basándose en una taxonomía de privacidad de 5 niveles.
  • Astro Annotate - Overlay de anotaciones visuales para sitios de staging de Astro. Los clientes anotan elementos HTML directamente en el navegador. Las anotaciones se almacenan como JSON estructurado, legible por desarrolladores y LLMs.
  • Astro DID:WEB - Una integración de Astro para soporte de DID:WEB (DID Web). Te permite integrar fácilmente la funcionalidad de DID:WEB en tus proyectos de Astro, permitiéndote trabajar con identificadores descentralizados en un contexto web.
  • Astro Flyweb - Una integración de FlyWeb para Astro — auto-genera /.well-known/flyweb.json para que los agentes de IA puedan descubrir tu contenido estructurado.
  • Astro Inline Review - Un puente entre revisores humanos y agentes de programación para sitios de Astro. Navega tu sitio renderizado, selecciona texto, adjuta notas y exporta Markdown consciente de la ubicación que un agente de programación (Claude, Cursor, Copilot, etc.) puede ejecutar inmediatamente — sin buscar entre archivos fuente para explicar dónde está el problema.
  • AstroJS TypeSafe Routes - Enrutamiento con seguridad de tipos para Astro. Definiciones de rutas auto-generadas, enlaces tipados y constructores de URL.
  • Starwind UI - Una colección de más de 45 componentes hermosamente diseñados, animados y accesibles construidos puramente con Astro y JS vanilla.
  • Mannisto Astro i18n - Una alternativa flexible al i18n integrado de Astro, con enrutamiento de locale, detección y traducciones para sitios estáticos y SSR.
  • Arisakit Astro i18n - Una librería de internacionalización (i18n) con seguridad de tipos para el framework Astro, que proporciona localización fácil y gestión de traducciones para tus proyectos de Astro.
  • Otrodigital i18n Next - Un módulo completo de internacionalización para sitios estáticos de Astro 5. Proporciona enrutamiento de URLs traducidas, localización de contenido, carga de markdown multilingüe y traducciones basadas en i18next.
  • Astro Markdown for Agents - Una integración de Astro que sirve automáticamente Markdown a agentes de IA — incluyendo en el plan gratuito de Cloudflare.
  • Astro HTML Validate - Integración de Astro que valida el HTML generado con html-validate.
  • Astro Intl - Sistema de internacionalización simple y con seguridad de tipos para Astro, inspirado en next-intl.
  • Fast Comments Astro - Un widget de comentarios en vivo rápido y con todas las funciones para Astro, impulsado por FastComments.
  • Storybook Astro - El framework de Storybook soportado por la comunidad para Astro.
  • Hybrid Astro UI - Elementos de UI para Astro usando un modelo híbrido: Astro sin JS + mejora progresiva con web components.
  • Astro Pro JetBrains plugin - Un plugin completo de IntelliJ para desarrollo con Astro con soporte IDE premium.
