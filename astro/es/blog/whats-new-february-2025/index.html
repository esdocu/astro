Tips y Herramientas

Las últimas utilidades e integraciones creadas por la comunidad para ayudarte a construir con Astro. 🛠

  • Starwind UI - Un conjunto de componentes potentes y accesibles para tus proyectos de Astro estilizados con Tailwind CSS v4.
  • Zorn Player - Un reproductor de video para Astro con varias funciones personalizables.
  • Accessible Astro - Todo lo que necesitas para construir sitios web accesibles, rápidos y hermosos con Astro.
  • astro-obsidian-starlight-notes-template - Una plantilla de inicio de Starlight que usa tus notas de Obsidian para el contenido.
  • astro-teleport - Un componente ligero de Astro para renderizar contenido en una parte diferente del DOM usando un Web Component.
  • astro-typesafe-routes - Una integración de Astro para generación de URLs con seguridad de tipos y enrutamiento.
  • storyblock-loader - Un loader construido por la comunidad para Astro Content Layer que obtiene contenido de la API de Storyblok.
  • astro-qiita-loader - Un loader de Qiita para Astro, que te permite usar publicaciones de Qiita como colecciones de contenido.
  • Bit - Una herramienta CLI para scaffolding de proyectos de Astro + PocketBase.
  • static-only-collections - Especifica colecciones de contenido que solo se usarán en páginas estáticas para reducir el código construido del lado del servidor y mejorar el rendimiento de la página.
  • Starlight View Transitions Plugin - Añade view transitions entre documentos, nativas del navegador, a tu sitio Starlight con una línea en tu configuración de Astro.

… y algunos construidos con Astro:

  • StudioCMS - Un CMS dedicado para Astro DB, con un nuevo lanzamiento beta 8.
  • Tract Stack - Una plataforma de publicación y constructor de sitios web con Astro sin código.
  • GitHub Activity - Visualiza la actividad reciente de GitHub de cualquier usuario en una hermosa lista.
  • Mayo App - Una app de streaming de audio auto-hospedable y basada en web.
