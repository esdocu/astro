Weird Wide Web

Algunos sitios del último mes que muestran lo que puedes hacer con Astro y que son peculiares, innovadores, inspiradores, ¡o simplemente divertidos!

  • Ascii Moon - Visualiza y recorre las fases de la Luna día a día: renderizadas en arte ASCII.
  • Give Back Guide - Una plataforma que conecta a viajeros con proyectos y experiencias que tienen un impacto positivo en las personas, los lugares y el planeta. Su objetivo es facilitar a los viajeros encontrar y apoyar proyectos locales en sus destinos que se alineen con sus valores.

  • Deep Time Navigator - Visualiza eventos mundiales a través del tiempo profundo, con escala logarítmica, para que puedas ver cómo el presente (un día, una semana, un año) se desvanece suavemente hacia el pasado más profundo.
  • TIDS:LOGG - ¡Lleva un registro del tiempo que pasas con Astro! (¿Y aprendes un poco de sueco al mismo tiempo?) Un sitio sueco de seguimiento de tiempo construido con Supabase.
  • Passport Photo Wiz - Crea fotos de pasaporte conformes en segundos, directamente desde tu navegador.

Tips y Herramientas

Las últimas utilidades e integraciones creadas por la comunidad para ayudarte a construir con Astro. 🛠

  • Back Link - Un componente que te permite invertir la dirección de la animación de transición de vista al usar el client router de Astro.
  • Astro Validate Env - Una integración de validación de variables de entorno para Astro que te da control sobre cuándo/dónde validar las variables de entorno y funciona en todos los contextos.
  • Astro Medium loader - Un loader de Astro que obtiene y analiza feeds RSS de Medium por nombre de usuario. Soporta enlaces canónicos, caché opcional y estrategias de almacenamiento flexibles.
  • Astro content loader - Carga archivos .astro con la Content Layer API de Astro, igual que harías con otros formatos (ej. Markdown, MDX).
  • unDocs - Un framework de documentación construido con Astro donde puedes crear tus propios docs y playground de componentes donde controlas el diseño, los componentes y el estilo.
  • Stripe Astro loader - Obtiene datos de la API de Stripe y los usa en colecciones de Astro.
  • Astro Payload local loader - Carga y cachea datos de una instancia de Payload CMS durante tu build de Astro, usando la API local.
  • Astro Markdown prerender - Una integración de Astro que establece automáticamente route.prerender basándose en las propiedades del frontmatter de tus archivos Markdown.
  • Astro Loader Hashnode - Un content loader potente para integrar publicaciones de blog de Hashnode en tu sitio web de Astro usando la Content Layer API.
  • Astro TOC generator - Un generador de tabla de contenidos para Astro.js que funciona específicamente con contenido de blog de Sanity CMS.
  • Astro Swiper - Componente de Astro para Swiper, dedicado a slider / carrusel / photo swiper / slide.
  • Astro llms.txt - Una integración de Astro para generar archivos de documentación compatibles con IA.
  • Astro terminal themes - Convierte configuraciones de temas de terminal (Ghostty, Warp, iTerm2, etc.) en temas de Tailwind CSS para tu sitio web de Astro.
  • Astro Mermaid - Una integración de Astro para renderizar diagramas de Mermaid con cambio automático de tema y renderizado del lado del cliente.
  • Astro SEO plugin - Un plugin SEO de configuración cero para Astro.
  • Astro Strapi loader - Una integración de Astro que permite la carga fácil de datos desde la Strapi Content API.

… y algunos construidos con Astro:

  • Is it really foss? - Donde los proyectos son evaluados para ver si son tan libres y de código abierto como se anuncian.
  • What’s the time? - Un conversor de tiempo moderno que ayuda a los ciudadanos globales a trabajar, reunirse y planificar entre zonas horarias.
  • Online Calculator - Una calculadora online sencilla.
  • Callculation - Un conjunto completo de herramientas para calculadoras, conversores, generadores y herramientas con IA, todo online y gratis.
  • Generador de CSS clamp() - Crea tamaños de fuente y diseños fluidos con precisión.
