Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • Astro Editor - Lee tus esquemas Zod y transforma el frontmatter en formularios inteligentes.
  • 🌟 Changelogs de Starlight - Muestra los changelogs junto a la documentación de tu proyecto.
  • 🌟 Menú contextual de Starlight - Añade un menú contextual a tu documentación de Starlight para acceso rápido a contenido optimizado para IA.
  • 🌟 Alertas de GitHub para Starlight - Renderiza las alertas de GitHub como asides de Starlight.
  • 🌟 Iconos de plugins de Starlight - Añade iconos a Astro Starlight: barra lateral, bloques de código, árbol de archivos.
  • Astro Table of Contents - Genera e incluye automáticamente una tabla de contenidos (TOC) para tus páginas web.
  • Astro Simple Feature Flags - Una integración simple y con tipado seguro de feature flags para Astro, impulsada por la Content Layer API.
  • Astro Loader Obsidian - Un loader de contenido para Astro que te permite tratar tu vault de Obsidian como una colección de contenido nativa.
  • Astro LLMS Generator - Descubre y procesa automáticamente todas tus páginas de Astro para generar archivos de documentación optimizados.
  • Integración de Varlock para Astro - Te ayuda a integrar varlock en un proyecto de Astro.
  • Astro Mithril - Habilita el renderizado del lado del servidor y la hidratación del lado del cliente para tus componentes de Mithril.
  • Astro Selfie - Genera capturas de pantalla de páginas para mostrar como imágenes de Open Graph.
  • Astro Remove Comments - optimiza tu sitio estático reduciendo el tamaño del archivo y eliminando comentarios relacionados con el desarrollo
  • Inline - Trae beasties (una librería para inlinear el CSS crítico de tu app y cargar el resto de forma diferida) a tu proyecto de Astro.
  • Astro CNAME - Genera un archivo CNAME usando el valor configurado de site.
  • Astro Render Component - Renderizador de componentes de Astro plug-and-play para pruebas rápidas con configuración cero en cualquier entorno JS/TS tipo DOM.
  • Astro Dev Only - Elimina archivos de tu compilación estática de producción final. Útil para borradores u otras páginas solo para desarrollo.
  • Authproto - Autentica a los visitantes de tu sitio usando ATproto. Para Bluesky y más allá.
  • Astro Lit - Habilita el renderizado del lado del servidor y la hidratación del lado del cliente para tus elementos personalizados de Lit.
  • Astro AI LLMS TXT - Crea un resumen optimizado para IA de tus páginas HTML.
  • Astro Dynamic Header - Un componente de encabezado dinámico y responsive para proyectos de Astro que puede alternar entre estilos flotante y pantalla completa con soporte de navegación desplegable multinivel.
  • Astro Static Headers - Captura los encabezados y redirecciones durante el prerender y escribe configuración específica de plataforma para Cloudflare, Netlify y Vercel.
  • Astro Tailwind 4 auto reference - Inyecta automáticamente directivas @reference de tailwind en los bloques de estilo de componentes de Astro.
  • Astrojs Hyperapp - Componentes de Hyperapp estáticos/renderizados en el servidor con Astro.
  • Configuración de ESLint para Astro - Configura ESLint automáticamente para lintar archivos de Astro con sintaxis de TypeScript.
  • Astro Startup Code - Ejecuta código junto con un despliegue de NodeJS.
