Noticias del Equipo de Astro

¡Tenemos un nuevo Comité Directivo Técnico (TSC)!

Damos la bienvenida a los maintainers principales Emanuele Stoppa Florian Lefebvre y Matt Kane al TSC de Astro, uniéndose al miembro actual Matthew Phillips , para guiar la dirección técnica de Astro y asegurar un código base saludable por muchos años más.

Lee más sobre el TSC 2025

Además, Matt ha asumido el rol de líder del equipo del framework junto a Sarah Rainsberger , ¡nuestra actual líder del equipo de documentación!

Y también estamos muy felices de dar la bienvenida a nuestro más nuevo maintainer de Astro, @stramel.

Historia feel-good de open source

El "What's New?" del mes pasado incluyó un post de blog de Angelika Cathor sobre las implicaciones del streaming de HTML en Astro y detalló el workaround que están usando actualmente.

En respuesta, el miembro del equipo principal de Astro Luiz Ferraz no solo identificó un problema en un servicio de terceros y contribuyó con una corrección de bug, sino que también creó una integración de Astro que te permite desactivar el streaming de Astro que destacamos abajo en la sección de Tips y Herramientas de este mes.

Tips y Herramientas

Las últimas utilidades e integraciones creadas por la comunidad para ayudarte a construir con Astro. 🛠

  • Astro Tanstack - Construye una Single Page Application (SPA) con Astro, TanStack Router y Query.
  • cfetch - Una integración de Astro que proporciona una función fetch cacheable para Astro SSR.
  • Astro WordPress - Construye temas de WordPress usando Astro.
  • Extensión Nova IDE para Astro - Proporciona resaltado de sintaxis y autocompletado básico para archivos .astro en el IDE Nova de Panic.
  • 🌟 Starlight Fullview Mode - Un plugin minimalista que colapsa las barras laterales de Starlight y expande el contenido principal al ancho completo.
  • Inox Tools: cut short - Una integración de Astro que te permite detener el procesamiento de una petición al instante y enviar una respuesta personalizada, ahora con una opción disableStreaming.
  • Typed Rest Routes - Una integración de Astro que permite crear endpoints de servidor con tipado seguro.
  • 🌟 Starlight scroll to top - Añade un botón de scroll hacia arriba a tu sitio web de documentación.
  • Astro og - App de la barra de herramientas de desarrollo de Astro que previsualiza tu página en plataformas sociales y muestra sus datos de open graph.
  • Astro Google tag manager - Una integración para inyectar snippets de Google Tag Manager en proyectos de Astro, soportando herramientas populares de analítica web.

… y algunos construidos con Astro:

  • Starlight Changelog - una versión bonita del propio CHANGELOG.md de Starlight, actualizada automáticamente con cada nuevo lanzamiento. ¡Márcalo como favorito para mantenerte al día con los cambios de Starlight! Construido con el tema Starlog de Astro y el astro-loader-github-releases content loader
  • Tract Stack - Un constructor de sitios web sin código que transforma tu presencia en línea de un folleto estático a una conversación dinámica que se adapta a cada lector, con un panel para entender las interacciones de tus usuarios. Construido con Astro y Turso.
