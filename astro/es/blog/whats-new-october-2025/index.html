¿Sabías que?

Astro ahora tiene un calendario de eventos en Luma para rastrear conferencias, meetups y otros eventos del Astroverse… ¡como Reuben Tier en React Advanced London en noviembre!

Si estás organizando un meetup local, una charla, un evento, o hablando en una conferencia, ¡avísale a Sarah Rainsberger ! No tenemos todos los detalles resueltos, ni los criterios específicos para publicar eventos externos. Pero sí queremos hacer que los eventos de Astro alrededor del mundo sean más descubribles. ¡Así que si estás dispuesto a probarlo, nosotros también!

Y el primer evento (pasado) en nuestro calendario de Luma fue el Astro Happy Hour en Ámsterdam, ¡patrocinado por Netlify! Gracias por salir a tomar algo, picar y tener grandes conversaciones. Se intercambiaron stickers, se (re)unieron conocidos, se hicieron nuevos amigos, se formaron alianzas y se tramaron planes.

¡Suscríbete a nuestro calendario de Luma y sabrás cuándo viene el próximo!

Tips y Herramientas

Las últimas utilidades e integraciones construidas por la comunidad para ayudarte a construir con Astro. 🛠

  • 🌟 Starlight Links - Extensión de Visual Studio Code que proporciona IntelliSense para enlaces en archivos Markdown y MDX para proyectos de documentación de Starlight.
  • 🌟 Mixedbread Search for Starlight - Una integración de Mixedbread para Starlight que reemplaza la búsqueda por defecto de Pagefind.
  • 🌟 Starlight DocSearch Typesense - Un plugin de Starlight que añade DocSearch impulsado por Typesense para búsqueda de documentación rápida y tolerante a errores tipográficos.
  • Astro Chatty GPT - Una API de chatbot simple para tu sitio de Astro. Impulsada por búsqueda de Upstash, GPT-5 y AI-SDK.
  • Astro ATProto OAuth Starter - Una plantilla starter mínima de Astro que demuestra autenticación OAuth con el AT Protocol (ATProto), el protocolo de red social descentralizado usado por Bluesky y otros servicios.
  • Astro Single File - Una integración que te permite empaquetar el CSS y el HTML en un solo archivo.
  • Astro Users - Una integración de Astro para gestión de usuarios con base de datos Cloudflare D1.
  • Expressive Code language badge - Un plugin para Expressive Code que añade insignias de lenguaje a los bloques de código con estilos personalizables.
  • Astro View Transition Script - Una función de utilidad para ejecutar un callback compatible con las view transitions de Astro.
  • Astro NProgress - Una integración para soportar la barra de progreso nanoscópica de NProgress.js que es compatible con las view transitions de Astro.
  • Astro Markdown Export - Una integración de Astro que genera archivos Markdown estáticos a partir de tus colecciones de contenido durante el build.
  • Astro Loader for Valise - Cargadores de contenido de Astro para el sistema de inventario de arte Valise, disponible tanto para obras de arte como para colecciones.
  • Astro Google Reviews - Incrusta el widget de reseñas de Google (impulsado por Wally) en tu sitio de Astro con un solo componente.
  • Astro Theme Toggler - Una utilidad simple, mínima y optimizada para alternar temas en Astro con JS vanilla.
  • Astro Dropdown - Una utilidad simple y personalizable de menú desplegable para Astro con solo JS vanilla.
  • Astro Shield - Una integración mínima que disuade a los scrapers automatizados mediante desafíos del lado del cliente sin requerir servicios externos.
  • Astro Plugin Media - Un plugin de medios para Astro que soporta la incrustación de múltiples reproductores de audio y video en Markdown.
  • Agent Summary - Una pequeña integración de Astro que convierte tu sitio construido en un catálogo legible por máquinas (AGENTS.md) para herramientas agénticas o basadas en LLM.

… ¡y algunos construidos con Astro!

  • Vista general de procedencia de npm - Visualiza rápidamente la seguridad de publicación para los paquetes npm de un maintainer.
  • 🌟 Temas de Starlight - El único lugar para previsualizar todos los temas de Starlight. ¿Tienes problemas para elegir entre Catppuccin, Flexoki, Rapide e Ion? ¡Encuentra todos tus temas favoritos de Starlight construidos por la comunidad disponibles en un solo sitio demo con un alternador de temas!
